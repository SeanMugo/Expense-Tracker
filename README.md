# 💰 Expense Tracker – Simple & Persistent

A lightweight, vanilla JavaScript expense manager built with **OOP principles**, **functional programming**, and **localStorage** for data persistence. Add, view, filter, and delete expenses – all with a clean, responsive UI powered by Tailwind CSS. Includes 

## ✨ Features

- ➕ **Add expenses** – with description, amount, and category (Food, Transport, Entertainment, Other)
- 📋 **View all expenses** – displayed in a sorted list (newest first)
- 🗑️ **Delete any expense** – one-click removal
- 🔍 **Filter by category** – instantly see expenses from a specific category
- 💶 **Today’s total** – automatically calculates the sum of all expenses added today
- 💾 **Persistent storage** – expenses are saved in `localStorage` and survive page refreshes
- 📱 **Responsive design** – works beautifully on desktop and mobile (Tailwind CSS)

## 🧰 Technologies Used

- **HTML5** – semantic structure
- **CSS3** + **Tailwind CSS** – utility-first styling
- **JavaScript (ES6+)** – classes, destructuring, spread/rest, array methods (`map`, `filter`, `reduce`)
- **Jest** – unit testing (4 core functions)
- **localStorage** – client-side data persistence
- **Git & GitHub Pages** – version control and deployment
